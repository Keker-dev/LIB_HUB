LIBHUB


У зарегистрированного пользователя два личных кабинета: читателя и писателя. В кабинете писателя он может просмотреть
свои произведения, изменить им цену(например, сделав бесплатными), выложить продолжение,
изменить статус работы и так далее. В кабинете читателя есть список последних прочитанных работ,
оставленных комментариев к главам, уведомления о выходе продолжений любимых книг. Незарегистрированный пользователь
не имеет кабинетов, но он может прочитать бесплатные произведения онлайн.
Зарегистрированный пользователь может получить роль администратора от другого администратора.
Администраторы следят за контентом на платформе.


Этапы:
1) Основная страница. +
    1.1) Кнопка профиля пользователя(для зарегистрированных). +
    1.2) Кнопки кабинетов читателя/автора пользователя(для зарегистрированных). +
    1.3) Уведомления сайта(о выходе продолжения любимых книг, для зарегистрированных). +
    1.4) Поиск книг. +
    1.5) Подборка популярных книг. +
    1.6) Кнопки регистрации/входа(для незарегистрированных). +
    1.7) Информация о сайте. +
2) Окно регистрации(открывается только для не вошедших в аккаунт пользователей). +
    2.1) Поля для ввода пароля, логина и почты. +
    2.2) Кнопка "Зарегистрироваться". +
3) Окно входа(открывается только для не вошедших в аккаунт пользователей). +
    3.1) Поля для ввода пароля, логина/почты. +
    3.2) Кнопка "Вход". +
4) Профиль пользователя. +
    4.1) Информация о пользователе. +
    4.2) Работы пользователя. +
    4.3) Кнопка редактировать(для владельца профиля). +
    4.4) Кнопка "Лайк"(не для владельца профиля). +
5) Кабинет читателя(для зарегистрированных). +
    5.1) Уведомления о продолжениях к произведениям. +
    5.2) 100 Последних просмотренных работ. +
    5.3) Понравившиеся книги. +
    5.4) Понравившиеся авторы. +
6) Кабинет автора(для зарегистрированных). +
    6.1) Уведомления о комментариях/лайках к произведениям пользователя. +
    6.2) Список работ пользователя. +
    6.3) Кнопка редактировать у каждой работы. +
    6.4) Кнопка добавить книгу. +
    6.5) Кнопка удалить книгу. +
7) Страница книги. +
    7.1) Название книги. +
    7.2) Автор книги. +
    7.3) Кнопка "читать". +
    7.4) Описание книги. +
    7.5) Кнопка лайка. +
    7.6) Кнопка добавить в понравившиеся(не для автора книги). +
    7.7) Кнопка "добавить главу"(для автора книги). +
    7.8) Кнопка "удалить главу"(для автора книги).
8) Страница главы книги.
    8.1) Навание книги. +
    8.2) Название главы. +
    8.3) Текст главы. +
    8.4) Комментарии к главе. +
9) Настройки(для зарегистрированных). +
    9.1) Настройки аккаунта. +
        9.1.1) Поле смены имени. +
        9.1.2) Поле смены пароля. +
        9.1.3) Поле смены информации о себе. +
        9.1.4) Кнопка выйти из аккаунта. +
        9.1.5) Кнопка удалить аккаунт. +
    9.2) Настройки текста. +
        9.2.1) Шрифт текста. +
        9.2.2) Размер текста. +
        9.2.3) Цвет текста. +
    9.3) Настройки API. +
        9.3.1) Отображение токенов пользователя и их срок действия +
        9.3.2) Кнопка "добавить токен". +
        9.3.3) Кнопка "удалить токен". +
    9.4) Остальное. +
        9.4.1) Поле для выбора, какие уведомления игнорировать. +
        9.4.2) Какое кол-во последних просмотренных работ отслеживать(до 100). +
        9.4.3) Кнопка очистить историю просмотров. +
10) Модернизация проекта по своему усмотрению.
